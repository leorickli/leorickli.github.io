My Projects

This section shows all my projects in Data Analysis and Data Engineering so far. You can find them hosted on GitHub and Kaggle.

Deploying WordPress on AWS

This is an excelent project to hone your AWS and cloud skills, since all cloud counterparts (GCP, Azure, etc) use similar structure compared to this. You will learn how to use VPCs, subnets, security groups, NAT gateways, route tables, VMs, file systems, databases, create domain names, SSL certificates and IAM roles, making it a great add to your portfolio.

Cyclistic BikeShare Data Analysis

I used BigQuery, Excel and Tableau to use the Google Data Analytics motto: Ask, Prepare, Process, Analyise, Share and Act.

Spotify ETL and Webscrapping Data Engineering

This is an ETL project using the Spotify API to get the daily top 50 global songs in the app. I used AWS to extract the file from a Jupyter notebook using Python to process into an S3 bucket, separating data between artist, album and songs into another S3 bucket, loading the data into AWS Athena for analysis using AWS Glue.

SportsStats Data Wrangling with SQL

I usesd Data Wrangling and Data Analysis skills with SQL deployed in Python using Pandas, Numpy and Matplotlib by creating a database with Data Governance concepts.

Data Engineering ETL Project

I made an ETL (Extract, Transform and Load) pipeline, commonly used in Data Engineering to extract the data from two json files that has a list of banks by market cap, transform them by multiplying it to a csv file that has the rates of all currencies referenced in USD, loading it into a simple csv file.
